# クイズ生成リクエスト

以下の条件で Google Cloud 認定資格の問題を1問生成してください。

## 試験情報

- **試験コード**: {{exam_code}}
- **ドメイン**: {{domain}}
- **セクション**: {{section}}
- **トピック**: {{topic}}
- **難易度**: {{difficulty}}

## 要件

1. 上記のトピックに関連した問題を作成してください
2. 4択の選択肢を用意してください
3. 正解は `correct` フィールドで 0-3 の数値で示してください（0=A, 1=B, 2=C, 3=D）
4. 解説では正解の理由と、他の選択肢が不正解である理由を説明してください
5. 日本語で問題と解説を作成してください

## 出力

JSON 形式のみで出力してください。マークダウンのコードブロックは使用しないでください。
