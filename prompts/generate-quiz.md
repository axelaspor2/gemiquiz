# クイズ生成リクエスト

以下の条件で Google Cloud 認定資格の問題を1問生成してください。

## 手順

1. まず `search_documentation` ツールで「{{topic}}」を検索してください
2. 検索結果から最も関連性の高い URL を選択してください
3. `read_documentation` ツールでドキュメントを取得してください
4. ドキュメント内容に基づいて問題を作成してください

## 試験情報

- **試験コード**: {{exam_code}}
- **ドメイン**: {{domain}}
- **セクション**: {{section}}
- **トピック**: {{topic}}
- **難易度**: {{difficulty}}

## 問題タイプ: {{question_type}}

{{question_type_instruction}}

## 要件

1. 上記のトピックに関連した問題を作成してください
2. 4択の選択肢を用意してください
3. 正解は `correct` フィールドで 0-3 の数値で示してください（0=A, 1=B, 2=C, 3=D）
4. 解説では正解の理由と、他の選択肢が不正解である理由を説明してください
5. 日本語で問題と解説を作成してください
6. ドキュメントから得た情報を元に、正確な内容の問題を作成してください

## 出力

JSON 形式のみで出力してください。マークダウンのコードブロックは使用しないでください。

```json
{
  "question": "問題文",
  "options": ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
  "correct": 0,
  "explanation": "解説文"
}
```
